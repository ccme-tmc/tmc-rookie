# TMC-Rookie

这个仓库存储了为帮助组内新同学学习理论计算科研工作所需工具而编写的简单教程. 

受编写者能力智识精力所限, 这些教程并不旨在给每一种工具做全面详细的介绍, 而是希望读者阅读后在面对这些工具时不至于感到不知所措.

## 初步大纲

  - [Linux基础](linux-basics/)
  - [编程技巧](programming)
    - Bash shell
    - Python
    - 版本控制系统
  - [计算软件的基本使用](softwares)
    - Gaussian
    - VASP
    - WIEN2k
    - GAP2
  - 计算结果可视化
    - XmGrace
    - Matplotlib
  - 论文写作
    - LaTeX
    - 文献管理
